# Virtual CV Portfolio

## Overview

This repository contains my personal portfolio website, developed as part of my BSc Information Technology studies. The website serves as a virtual CV, showcasing my education, technical skills, projects, certifications, achievements, and contact information.

## Live Website

**Portfolio:** https://viv120.github.io/Virtual-CV/

## Features

* Responsive design for desktop and mobile devices
* Modern user interface with a red and black colour scheme
* About Me section
* Education and Academic Transcript
* Technical Skills
* Project showcase with GitHub repository links
* Work Experience
* Certifications with preview modal
* Achievements
* Contact information
* Downloadable CV
* Smooth scrolling navigation
* Back-to-top button

## Technologies Used

* HTML5
* CSS3
* JavaScript
* GitHub Pages

## Running the Project

1. Clone the repository:
```bash
git clone https://github.com/Viv120/Virtual-CV.git
```

2. Open the project folder.

3. Open `index.html` in your preferred web browser.

4. Alternatively, view the live portfolio at:
   https://viv120.github.io/Virtual-CV/

No additional software or installation is required.
